# Open BK Web

Scaffold frontend React + TypeScript + Vite dengan:
- TanStack Router (file-based)
- TanStack Query (Suspense-first)
- MUI v7 theme system

## Start

```bash
npm install
npm run router:generate
npm run dev
```

Catatan:
- `src/routeTree.gen.ts` dibuat otomatis oleh `@tanstack/router-plugin`/`tsr`.
- Route smoke target awal: `/`, `/nis-gate`, `/send-letter`, `/admin/inbox`.
